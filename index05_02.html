querySelctor() method UseCase

querySelctor() method can be used to choose the 1st instance or specific nodes of the DOM

eg
typing in console > document.querySelctor(".masthead") , this looks for a node element with class masthead

the same can be done for id's i.e use #masthead inside the parantheses.

Now, using document.querySelectorAll("a") method, we shall obtain an HTML collection/ node tree of anchor tag instances.
The same querySelectorAll() method, can be used to look out for elements used often in the document.
WhereAs, querySelctor() method, is used to target specific elements as earlier above.

document.querySelector(".menu .has-children a"), returns the first link instance.
inside the element with the class has-children, inside an element with the class menu.

furthermore
document.querySelectorAll(".social-nav a[href*='linkedin.com']") can give us any link pointing to linkedin inside the element with a class social-nav.

Making numerous queries using different selectors separated by a comma

document.querySelctorAll(".social-nav li a, .menu li a"), returns all links inside the li, inside either in element with class menu or social-nav

these methods can be used in areas where old methods eg getElementById()
