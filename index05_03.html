document.querySelector(".main-title").innerHTML can be used to access the innerHTML property inside the element with class main-title.
if it's a value, then it's directly returned.
‐ output: "Welcome to Moonwalk Manor"
document.querySelector(".main-title").outerHTML, can be used to access the entire HTML property and in this case is the element. it includes the HTML wrapper itself.
‐ Output: "

Welcome to Moonwalk Manor.

"

changing the value of properties[inner and outer HTML]

document.querySelector(".main-title").innerHTML = "All your headings belong to us!"
‐ Output: "All your heaings belong to us!"

Same can be achieved with id's

eg document.querySelector("#showCase").id = "slideShow", hence the id in the div container is changed.
